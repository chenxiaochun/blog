# Chrome Dev Tools

打开开发工具快捷键：

```
F12
```

或者

```
Command/Ctrl + Alt + J
```

## 命令菜单

打开方式快捷键：

```
Command/Ctrl + Shift + P
```

|命令名称|作用
---|---
`Capture node screenshot` | 对选中的当前 dom 节点截图
`Capture full size screenshot` | 全屏截图。如果当前页面有滚动条，它会自动往下滚动截图
